### upd 1：更新了 fun mode，规则和 ICPC 赛制不一样，fun mode 参考了 cf 的积分赛制，每题计分规则如下：

假设某队伍在 $t$ 时刻（分钟）通过某题，且之前不算编译错误的错误提交次数为 $x$ ，那么本题得分为 $\max(600-t-10x^2,200)$。该计分方式的出发点是侧重考察队伍代码的一次 AC 率。
